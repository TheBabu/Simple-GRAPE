# Simple GRAPE
Test simple GRAPE algorithm

## Installation
1. Create virtual environment
   
   ```
   virtualenv venv
   source venv/bin/activate
   ```
2. Install
   
   ```
   pip install -e .
   ```
   
3. Run scripts
   ```
   ./[path_to_script]/main.py
   ```

## Installation (with poetry)
1. Install
   
   ```
   poetry install
   ```
2. Run scripts
   
   ```
   poetry run ./[path_to_script]/main.py
   ```
   or alternatively
   ```
   poetry shell
   ./[path_to_script]/main.py
   ```
